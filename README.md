## 🎯 시스템 개요

이 시스템은 **TBM(Tool Box Meeting) 안전회의를 자동화**하는 B2B 서비스입니다.

### 주요 기능
| 기능 | 설명 |
|------|------|
| 🎤 음성 녹음 | 회의 내용을 말하면 자동으로 텍스트로 변환 |
| 🤖 AI 분석 | Claude AI가 위험요인을 자동 식별 및 분류 |
| 📊 위험성평가 | 빈도/강도 기반 위험도 자동 계산 |
| 👥 참석자 관리 | 모바일 서명, 실시간 참여 현황 |
| 🌏 다국어 지원 | 한국어, 영어, 태국어, 베트남어 |

### 파일 구조
```
tbm-system/
├── backend/                 # 서버 (Node.js)
│   ├── server.js           # 메인 서버 파일
│   ├── package.json        # 의존성 목록
│   └── .env.example        # 환경 설정 샘플
├── frontend/               # 화면 (웹페이지)
│   └── index.html          # 통합 프론트엔드
├── database/               # 데이터베이스
│   └── schema.sql          # 테이블 구조
└── README.md               # 이 파일
```

## 💻 설치 방법

### 1단계: Node.js 설치
> Node.js는 서버를 실행하는 데 필요한 프로그램입니다.

1. [nodejs.org](https://nodejs.org) 접속
2. "LTS" 버전 다운로드 (권장)
3. 설치 프로그램 실행 → "다음" 클릭하며 설치 완료

**설치 확인:**
```bash
node --version   # v18.x.x 이상이면 OK
npm --version    # 9.x.x 이상이면 OK
```

### 2단계: 프로젝트 다운로드
```bash
# 프로젝트 폴더로 이동
cd tbm-system

# 필요한 패키지 설치
cd backend
npm install
```

### 3단계: 환경 설정
```bash
# 설정 파일 복사
cp .env.example .env

# 편집기로 .env 파일 열기 (메모장, VS Code 등)
# 아래 값들을 실제 값으로 변경
```

**.env 파일 주요 설정:**
```
# Claude API 키 (필수) - anthropic.com에서 발급
CLAUDE_API_KEY=sk-ant-여기에키입력

# 네이버 클라우드 API (선택)
CLOVA_SPEECH_CLIENT_ID=여기에입력
CLOVA_SPEECH_CLIENT_SECRET=여기에입력
```

## 📱 기능 설명

### 1. 음성 녹음 탭
- **마이크 버튼**: 클릭하면 녹음 시작/중지
- **녹취록**: 말한 내용이 실시간으로 텍스트로 변환
- **AI 분석**: 녹취록을 분석하여 위험요인 자동 추출

### 2. 기본 정보 탭
- 날짜, 시간, 장소 등 TBM 기본 정보 입력
- **작업 내용**: AI 분석에 활용됨

### 3. 보호구 탭
- 체크리스트 형식으로 필수 보호구 확인
- 안전모, 안전화, 안전장갑, 보안경 등

### 4. 참석자 탭
- 참여자 목록 및 서명 관리
- 터치/마우스로 서명 가능

### 5. 위험성평가 탭
- AI가 추출한 위험요인 목록
- 빈도(1-5) × 강도(1-4) = 위험점수
- 12점 이상: 🔴 고위험, 6점 이상: 🟡 중위험

## ☁️ 서비스 배포 (네이버 클라우드)

### 필요한 서비스

| 서비스 | 용도 | 예상 비용 |
|--------|------|----------|
| Server | 웹 서버 | 월 ~5만원 |
| Cloud DB for MySQL | 데이터 저장 | 월 ~5만원 |
| Object Storage | 파일 저장 | 월 ~3천원 |
| CLOVA Speech | 음성→텍스트 | 분당 ~15원 |
| CLOVA Voice | 텍스트→음성 | 1000자당 ~4원 |

### 배포 순서

1. **네이버 클라우드 콘솔 접속**
   - console.ncloud.com 가입/로그인

2. **Server 생성**
   - Products → Compute → Server
   - Ubuntu 22.04, Standard (2vCPU, 4GB)

3. **데이터베이스 생성**
   - Products → Database → Cloud DB for MySQL
   - `database/schema.sql` 실행

4. **서버에 코드 배포**
   ```bash
   # 서버 접속 후
   git clone [프로젝트URL]
   cd tbm-system/backend
   npm install
   npm start
   ```

5. **도메인 연결**
   - DNS 설정으로 도메인 연결
   - SSL 인증서 적용

## 🆘 문의 및 지원

### 자주 묻는 질문

**Q: 서버가 실행되지 않아요**
- Node.js가 설치되어 있는지 확인
- `npm install`을 실행했는지 확인
- 3000번 포트가 다른 프로그램에서 사용 중인지 확인

**Q: 음성 인식이 안 돼요**
- Chrome 브라우저 사용 권장
- 마이크 권한 허용 확인
- HTTPS 환경에서만 정상 작동 (배포 시)

**Q: AI 분석이 안 돼요**
- `.env` 파일에 Claude API 키가 있는지 확인
- API 키가 유효한지 확인 (console.anthropic.com)
- 데모 모드에서는 시뮬레이션 데이터 사용
